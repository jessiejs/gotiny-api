API Documentation

Welcome to the GoTiny API!

With the GoTiny API, you can generate short links for your projects with just a few lines of code. The endpoint the API uses is:

https://gotiny.cc/api

When it comes to code, less means more. That’s why we set out to create the easiest to use API we could imagine. To create a short link, all you gotta do is send a POST request to our endpoint, with the long url in the body. The request body should be a JSON, formatted as follows:

POST https://gotiny.cc/api
{
  "long": "https://amazon.com/very-long-url"
}

That’s it! The response you’ll receive will be a json object with 4 properties: the original url, the GoTiny code, the shortened url and a version of the shortened url that includes the http-protocol, for easier linking. This way, you can present a clean and easy to read link to your users, while making sure their devices will handle the url correctly.

{
  "long": "https://amazon.com/very-long-url",
  "code": "y68hxc",
  "tiny": "gotiny.cc/y68hxc",
  "link": "https://gotiny.cc/y68hxc"
}

Note that you’re not required to use the long attribute in your code. GoTiny will automatically resolve the short link in the users web browser to point to the original url.
